Робота присвячується розробці програмного продукту, а саме "Система керування автосалоном Toyota", яка була розроблена для швидкої взаємодії з базою даних.
Метою виконання курсового проекту є набуття практичних навичок проектування баз даних та розробки програмного забезпечення з управління базами даних із використанням СУБД. 
У ході виконання даного курсового проекту необхідно розробити інформаційну базу даних для менеджерів, які взаємодіють із клієнтами та автомобілями.
